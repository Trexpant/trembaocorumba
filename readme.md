# 🚂 Trem Bão - Doceria Artesanal

Site institucional e de divulgação para a **Doceria Trem Bão**, localizada em Corumbá-MS. O projeto foi desenvolvido com foco em conversão, design apetitoso e facilidade de contato via WhatsApp.

## ✨ Funcionalidades

- 📱 **Design 100% Responsivo:** Layout adaptável para celulares, tablets e desktops.
- 🎬 **Hero com Vídeo:** Brigadeiro girando em alta qualidade, substituindo animações pesadas.
-  **Botões de Ação Direta:** Links otimizados para WhatsApp (`67 98406-0550`) e Instagram (`@trembao`).
- 🍰 **Galeria de Produtos:** Grid moderno pronto para receber fotos reais dos doces.
- 📍 **Informações Locais:** Endereço completo, horários e mapa de contato.
- 🌊 **Animações Suaves:** Efeitos de fade-in ao rolar a página e transições elegantes.

## 🛠 Tecnologias

- **HTML5** & **CSS3** (Grid, Flexbox, Variáveis CSS)
- **JavaScript** (Vanilla, sem dependências)
- **Font Awesome 6** (Ícones vetoriais)
- **Google Fonts** (Playfair Display & Poppins)
- *(Opcional)* Three.js foi removido em favor do vídeo para melhor performance e carregamento.
